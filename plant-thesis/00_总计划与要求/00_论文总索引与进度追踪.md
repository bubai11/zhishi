## 四、格式规范速查（来自模板）

| 格式项 | 规范 |
|---|---|
| 一级标题 | 小三号黑体加粗，如：`1 绪论` |
| 二级标题 | 四号宋体，如：`1.1 研究背景与意义` |
| 三级标题 | 小四号宋体，如：`1.1.1 国内研究现状` |
| 正文 | 小四号宋体，1.5倍行距 |
| 图题 | 宋体五号，居中，图下方，如：`图1.1 系统总体架构图` |
| 表题 | 宋体五号，居中，表上方，如：`表3.1 功能测试用例表` |
| 参考文献引用 | 右上角方括号，如：`[1]`，`[3-5]` |
| 参考文献格式 | GB/T 7714，期刊[J]、著作[M]、学位论文[D]、电子文献[EB/OL] |
| 正文字数 | 8000字以上（不含图表） |
| 参考文献数量 | 不少于15篇，英文不少于2篇 |
| 页眉 | 左：广东金融学院；右：本科毕业论文（设计）+论文题目 |
| 页码 | 摘要~目录：罗马数字；正文起：阿拉伯数字 |
