# 性能测试操作说明与记录

> 对应论文第六章 6.3 性能测试

---

## 测试方法

使用 **Postman** 对主要接口各发送10次请求，在 Tests 面板记录响应时间，取平均值和最大值。

---

## Postman操作步骤

1. 打开系统（`start.bat`），确认前后端都在运行
2. 打开Postman，新建Collection命名为"植物系统性能测试"
3. 按下表逐一添加请求，每个请求发送10次，记录响应时间（Postman右侧状态栏显示 `XXX ms`）

---

## Lighthouse前端性能报告

操作步骤：
1. 在Chrome中访问 `http://localhost:3000`
2. 按F12打开DevTools → 点击顶部"Lighthouse"标签（如没有，点击右侧"≫"找）
3. Categories选：Performance、Accessibility、Best Practices、SEO
4. Device选：Desktop
5. 点击"Analyze page load"，等待约30-60秒
6. 截图整个报告存为 `10_系统截图/Lighthouse性能报告.png`
7. 记录以下指标到论文：

| 指标 | Lighthouse评分/值 |
|---|---|
| Performance | /100 |
| First Contentful Paint (FCP) | s |
| Largest Contentful Paint (LCP) | s |
| Speed Index | s |
| Accessibility | /100 |
